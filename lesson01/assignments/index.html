Using ES6+ Methods

FOREACH | Array

MAP | Enter a number to add to each number in the array.

FILTER | Enter a number to filter the array by.

This will create a new array with only numbers from the original array that are greater than (>) the number you enter here.

REDUCE | Enter a number to add to the array and then see the sum.

FIND | Enter a number to find the first number in the array that is greater.

SOME | Enter a number to find all numbers in the array that are less than.

Every | Enter a number and see if it is greater than all the numbers in the provided array.
